# Final Project ( To-do list App )

## Overview
In this project, we will be mainly using useState + useEffect.

## Assignment Requirements
- Minimum of 5 distinct pages/views within the application to demonstrate routing and component organization.
- Use React context management to manage global state (e.g., user data, settings, or app-wide data).
- Effectively implement React hooks: useEffect for side effects and useState for component state management.
- Implement data persistence at least at the browser level (e.g., localStorage, IndexedDB) so user data is saved between sessions.
- Deploy the app live on the internet using platforms such as Vercel, Netlify, GitHub Pages, or similar.
- The app must achieve at least 90% of the overall planned functionality for the chosen project.
- Design a user-friendly interface with good UX/UI practices, including responsive design and intuitive navigation.

## Deliverables
- Deployed application URL accessible publicly on the internet.
- Source code repository with clear README instructions.
- A brief report (1-2 pages) describing the approach, challenges faced, and how key React concepts (context, hooks) are used.

## Evaluation Criteria
- Completeness and correctness of functionality (90%+ target)
- Use of React context and hooks appropriately
- Data persistence implementation
- UI/UX design quality and responsiveness
- Code organization, clarity, and documentation
- Live deployment accessibility
